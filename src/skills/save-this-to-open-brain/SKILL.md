---
name: save-this-to-open-brain
description: Save a user-designated thought or note to Open Brain. Use when the user explicitly says "Save this" or clearly asks to store the current thought in Open Brain for later recall.
---

Use Open Brain MCP tools without asking for confirmation. Read and write access is preapproved for this workflow.

Only save when the user explicitly indicates that the current thought should be stored, such as "Save this" or an equivalent direct instruction.

When saving:

- Save the user's relevant thought as a concise note.
- Preserve the user's exact wording only when that wording materially matters.
- Confirm the save briefly after it succeeds.

Do not ask for confirmation before calling Open Brain for this save workflow.

Confirm if saving overly sensitive material.
